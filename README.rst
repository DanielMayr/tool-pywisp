=====================================================
PyWisp - Weird visualisation of test bench prototypes
=====================================================


Install
-------

For standard use:

.. code-block:: bash

    $ git clone pywisp
    $ cd pywisp/
    $ python setup.py install

For development use:

.. code-block:: bash

    $ virtualenv pywisp
    $ git pywisp
    $ cd pywisp/
    $ python setup.py develop
